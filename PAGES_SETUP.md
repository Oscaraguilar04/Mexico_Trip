# Publish the trip site (one-time setup)

The site is ready to go live on **GitHub Pages**. Oscar needs to flip one switch in GitHub (takes ~30 seconds), then the link works for everyone.

## Step 1 — Enable GitHub Pages

1. Open **https://github.com/Oscaraguilar04/Mexico_Trip/settings/pages**
2. Under **Build and deployment → Source**, choose **GitHub Actions**
3. Save if prompted

## Step 2 — Deploy

1. Open **https://github.com/Oscaraguilar04/Mexico_Trip/actions**
2. Click **Deploy site to GitHub Pages**
3. Click **Run workflow** → **Run workflow**

Wait about 1 minute. The site will be live at:

**https://oscaraguilar04.github.io/Mexico_Trip/**

## Step 3 (optional) — Shorter, more memorable URL

Rename the repository to `colima-2027`:

1. Open **https://github.com/Oscaraguilar04/Mexico_Trip/settings**
2. Under **Repository name**, change `Mexico_Trip` → `colima-2027`
3. Confirm

The new share link becomes:

**https://oscaraguilar04.github.io/colima-2027/**

That is the link to text to Oscar, Leah, Raul, Rosa, Lily, Mike, and Yolee.

Copy the live link from the **Share with the group** section on the home page, or use the URL above directly.
