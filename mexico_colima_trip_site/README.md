# Colima February Trip Guide Website

Open `index.html` in a browser. The site is static and responsive.

Important: the savings tracker is a front-end prototype. It stores contributions in the browser with localStorage and can create a shareable snapshot link. It does not move or hold real money. For real group savings, connect the UI to a backend such as Supabase/Firebase and a payment flow such as Stripe, Zelle, Cash App, Venmo, or a shared bank account workflow.
